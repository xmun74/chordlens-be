## 1. 배경 및 목적

### 1-1. 배경

기타를 독학하는 사람들은 유튜브에서 커버 영상이나 레슨 영상을 찾아보지만, 실제 코드를 파악하려면 악보를 별도로 구매하거나 영상을 반복 재생하며 코드를 일일이 받아 적어야 한다. 이 과정은 번거롭고 초보자에게 특히 높은 진입 장벽이 된다.

### 1-2. 목적

YouTube URL 하나를 입력하면 기타 코드를 자동으로 추출하고, 시각적 다이어그램으로 바로 배울 수 있는 학습 도구를 만든다. 별도의 악보 구매나 코드 받아쓰기 없이 원하는 곡을 바로 연습할 수 있게 한다.

### 1-3. 목표 지표

| 지표                | 목표                                       |
| ------------------- | ------------------------------------------ |
| 월 운영 비용        | 10,000원 이하                              |
| 코드 분석 소요 시간 | 3분 영상 기준 20초 이내                    |
| MVP 출시            | 1단계 (로컬 구동) → 2단계 (배포) 순차 진행 |

## 2. 사용자 정의

### 2-1. 타겟 사용자

기타를 독학 중인 입문자 ~ 초중급자. 악보를 읽기 어렵고 유료 레슨 부담이 있는 20~30대.

### 2-2. 사용자 문제

| 문제                           | 현재 해결 방식               | 불편함                       |
| ------------------------------ | ---------------------------- | ---------------------------- |
| 원하는 곡의 코드를 모름        | 악보 구매, 유튜브 검색       | 유료이거나 정확하지 않음     |
| 유튜브 레슨 영상에서 코드 파악 | 영상 반복 재생하며 코드 메모 | 시간이 오래 걸림             |
| 기타 코드 모양을 모름          | 구글 이미지 검색             | 영상-악보-검색 왔다갔다 불편 |

## 3. 핵심 기능 (Feature)

### Feature 1. YouTube URL 입력 및 오디오 분석

사용자가 YouTube URL을 붙여넣으면 서버에서 오디오를 추출하고 기타 코드를 분석한다.

**요구사항:**

- YouTube URL 형식 유효성 검사 (`youtube.com/watch?v=`, `youtu.be/` 두 형식 허용)
- 분석 진행 상태를 단계별로 표시한다: `오디오 추출 중 → 코드 분석 중 → 완료`
- 동일 URL 재요청 시 DB 캐시에서 즉시 반환한다 (재분석 없음)
- 분석 실패 시 원인을 알 수 있는 에러 메시지를 표시한다

**제외 범위:**

- 비공개 영상, 연령 제한 영상은 지원하지 않는다
- 장조/단조 이외의 코드(7th, sus4, dim 등)는 v0.1에서 지원하지 않는다

---

### Feature 2. 기타 코드 타임라인

분석된 코드를 시간 순서대로 가로 타임라인으로 표시한다.

**요구사항:**

- 각 코드 칩에 시간과 코드명을 표시한다 (예: `00:12 Am`)
- 코드 칩을 클릭하면 해당 코드가 아래 다이어그램 그리드에서 강조된다
- 타임라인은 가로 스크롤을 지원한다

---

### Feature 3. 기타 코드 다이어그램

분석된 코드를 기타 다이어그램(코드 폼) SVG로 시각화한다.

**요구사항:**

- 곡에 등장한 코드를 중복 없이 그리드로 배치한다
- 각 다이어그램 카드에 코드명과 운지 위치를 표시한다
- 모바일 2열, 태블릿 3열, 데스크탑 4열 그리드로 반응형 배치한다
- 타임라인에서 선택한 코드를 강조 표시한다

---

### Feature 4. 결과 공유 링크

분석 결과를 URL로 공유할 수 있다.

**요구사항:**

- 결과 페이지 고유 URL을 생성한다 (`/result/:id`)
- 공유 버튼 클릭 시 URL을 클립보드에 복사한다
- 공유 링크로 접근 시 OG 태그(제목, 썸네일)가 포함된다

---

## 4. 기능 범위 정의

### v0.1 MVP (1단계 — 로컬)

- MP3 직접 업로드 → Basic Pitch + Tonal.js → 코드 다이어그램
- 서버 없이 브라우저에서 전부 처리

### v0.2 (2단계 — 배포)

- YouTube URL 입력 지원
- Railway 서버 (yt-dlp + autochord) 연동
- Supabase 캐싱 + 결과 공유 링크

### v0.3 이후 (백로그)

- 코드 카드 클릭 시 운지 방법 상세 팝업
- 강제 재분석 옵션
- 즐겨찾기 / 저장 기능

## 5. 사용자 시나리오 (User Story)

```
시나리오 1 — 기본 분석

사용자가 배우고 싶은 곡의 YouTube URL을 입력한다.
"코드 분석하기" 버튼을 누른다.
로딩 화면에서 단계별 진행 상태를 확인한다.
분석이 완료되면 영상 정보(제목, 채널명, 썸네일)와 함께
타임라인과 코드 다이어그램이 표시된다.
사용자는 코드 다이어그램을 보며 기타를 연습한다.
```

```
시나리오 2 — 캐시 활용

사용자가 이미 분석된 URL을 다시 입력한다.
로딩 없이 즉시 결과가 표시된다.
```

```
시나리오 3 — 결과 공유

사용자가 결과 페이지에서 "공유" 버튼을 누른다.
URL이 클립보드에 복사된다.
복사한 링크를 친구에게 전달한다.
친구가 링크를 열면 동일한 결과 페이지가 표시된다.
```

## 9. 제약 사항

| 항목              | 내용                                                        |
| ----------------- | ----------------------------------------------------------- |
| YouTube ToS       | yt-dlp를 이용한 오디오 추출은 개인 프로젝트 수준에서만 운영 |
| 코드 인식 범위    | 장조 12개 + 단조 12개 = 24개 코드만 지원 (v0.1 기준)        |
| 비공개 영상       | 분석 불가, 명확한 에러 메시지 표시                          |
| Railway 무료 한도 | 월 500시간. 초과 시 $5/월 플랜 전환 필요                    |
